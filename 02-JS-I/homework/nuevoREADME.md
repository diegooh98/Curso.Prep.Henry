**QUE ES UNA VARIABLE?**

1. `Una variable es donde se guarda y se recupera) datos que se utilizan en un programa.`

Cuando escribimos código, las variables se utilizan para:

Guardar datos y estados.
Asignar valores.
Representar valores dentro de una expresión matemática.
Mostrar valores por pantalla.

EJEMPLO: 
 animal ---> es la varible
 gato------> es el dato
 
 entonces tenemos:
 animal   = gato

 EJEMPLO: 
 numero ------> es la varible
 10     ------> es el dato

entonces tenemos:
  numero = 10

**QUE ES UN STRINGS?**

2. `Los strings son bloques de texto que siempre van entre comillas.`

EJEMPLO:
colores ------> es la varible 
"azul"    ------> es el strings

entonces tenemos:
colores = "azul"

**QUE SON LAS FUNCIONES?**

3. `Las funciones son tipos de objetos para poder hacer o modificar algo con las varibles.`

EJEMPLO:
 function numeros() {
     var unNumero = 50
 }
esto es una funcion y dentro de ella hay una variable. (unNumero = 50)

ANATOMIA DE UNA FUNCION:
function numeros() {

}

primero empezamos con la palabra function, 
luego el nombre de la funcion que es "numeros"
despues abrimos y cerramos parentesis
y por ultimo abrimos y cerramos llaves

//////////////////////////////////////////////////////////

**QUE SON LAS DECLARACIONES?**

4. `Las declaraciones es cuando queremos devolver o mostrar el resultado de una funcion:`

EJEMPLO:
 function numeros() {
     var unNumero = 50 + 50
    
     return unNumero;

 }

 .el resultado de la suma de los dos numeros 50 y 50, es lo queremos mostrar y lo hacemos con la palabra (return)
 
 La palabra: (return) es la declaracion y viene del ingles retornar o devolver.


//////////////////////////////////////////////////////////

**QUE SON LOS VALORES BOOLEANOS (TRUE, FALSE)?**

5. `los valores booleanos (true y false) es cuando queremos afirmar o negar algo`

EJEMPLO:
perro = animal
return true;

aqui estamos preguntado si un perro es un animal, entonces retornamos o mostramos con la palabra (true) que es verdad, la palabra true, significa en ingles verdadero.

o en caso contrario:
EJEMPLO:
perro = pelota
return false;

aqui estamos preguntando si un perro es igual a una pelota, entonces le decimos que no con la palabra false, ya que false en ingles significa "falso"

recuerden que siempre debe ir la palabra "return" que significa devolver o mostrar y luego ponemos "true o false" que significa verdadero o false, para decir si es afirmativo o falso.
